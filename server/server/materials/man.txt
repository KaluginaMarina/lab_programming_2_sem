remove_last: удалить последний элемент из коллекции
load: перечитать коллекцию из файла
info: вывести в стандартный поток вывода информацию о коллекции (тип, дата инициализации, количество элементов и т.д.)
remove_greater {element}: удалить из коллекции все элементы, превышающие заданный. Для завершение ввода дважды нажмите <enter>
add_if_max {element}: добавить новый элемент в коллекцию, если его значение превышает значение наибольшего элемента этой коллекции. Для завершение ввода дважды нажмите <enter>
add_if_min {element}: добавить новый элемент в коллекцию, если его значение меньше, чем у наименьшего элемента этой коллекции. Для завершение ввода дважды нажмите <enter>
add {element}: добавить новый элемент в коллекцию. Для завершение ввода дважды нажмите <enter>
help / ?: открыть справку
print: вывести краткую информацию об элементах коллекции
exit / ctrl + D: выход
